## Style

Never use barrel imports.

## Content

Make use of Astro Starlight's built-in markdown components like Asides (https://starlight.astro.build/guides/authoring-content/#asides) and Footnotes..

#### Aside

Aside blocks are indicated using a pair of triple colons ::: to wrap your content, and can be of type note, tip, caution or danger. Generally you should only use :::note, or :::tip[Did you know?] for info that is less examinable.

```markdown
:::note
Content
:::
```
