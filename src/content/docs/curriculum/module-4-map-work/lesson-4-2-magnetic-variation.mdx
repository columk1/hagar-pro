# Lesson 4.2: Magnetic Variation

Understanding the difference between **True Heading** and **Magnetic Heading** is essential for navigation. This lesson covers how to convert between them using magnetic variation.

## Key Concepts

### True North vs. Magnetic North

- **True North (TN)**: The geographic north pole — the point where Earth's axis of rotation meets the surface at 90°N latitude.
- **Magnetic North (MN)**: The location where Earth's magnetic field lines point vertically downward. This is where your compass needle points.

Magnetic North is not at the same location as True North. It moves over time and is currently located in the Arctic Ocean, northwest of Greenland.

### Magnetic Variation

**Magnetic Variation** (also called magnetic declination) is the angular difference between True North and Magnetic North.

- **East Variation**: Magnetic North lies east of True North. You must **subtract** east variation when converting from True to Magnetic.
- **West Variation**: Magnetic North lies west of True North. You must **add** west variation when converting from True to Magnetic.

### Memory Aid

> **"East is Least, West is Best"**
>
> - East variation → **subtract**
> - West variation → **add**

### The Formula

```
True Heading = Magnetic Heading ± Variation
```

Or rearranged to calculate Magnetic Heading:

```
Magnetic Heading = True Heading ∓ Variation
```

- For **East** variation: MH = TH − Variation
- For **West** variation: MH = TH + Variation

## Reading the VNC Chart

VFR Navigation Charts (VNCs) display **isogonic lines** — lines connecting points of equal magnetic variation. These lines are labeled with the variation value (e.g., "14°W" or "8°E").

- Find your location on the chart
- Identify the nearest isogonic line
- Note whether it's East or West variation
- Apply the memory aid when converting headings

## Quick Reference

| Variation Type | Formula       | Example           |
| -------------- | ------------- | ----------------- |
| 14° West       | MH = TH + 14° | TH 090° → MH 104° |
| 10° East       | MH = TH − 10° | TH 090° → MH 080° |

## Exam Tips

1. Always check the variation value on your VNC chart for the area you're flying in
2. Remember the memory aid: "East is Least, West is Best"
3. Double-check your math — a heading error can lead to significant navigation mistakes
4. Variation changes slowly over time (yearly updates are printed on charts)
